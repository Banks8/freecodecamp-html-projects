This repository contains beginner HTML projects completed as part of the FreeCodeCamp Responsive Web Design certification.
The focus of these projects is to build a strong foundation in semantic HTML, accessibility, and clean structure.

Projects included: hotel feedback form, a recipe page and a checkout page etc.
Skills practiced: Sementic HTML5 structure, Forms and input validation, Accessibility best practices and Page layout and content organization.

What I learned:
Through these projects, I developed a solid understanding of how to structure web pages using sementic HTML, create accessible forms, and organize content in a clean and readable way.
